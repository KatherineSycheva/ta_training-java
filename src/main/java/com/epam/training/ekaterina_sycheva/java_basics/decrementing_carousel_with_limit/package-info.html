Decrementing Carousel with limit

In this exercise class DecrementingCarousel was extended with subclass DecrementingCarouselWithLimitedRun.

This subclass decrements elements as a usual DecrementingCarousel.

The difference is that this implementation produces a carousel run, which limits number of calls to the next method.

When the limit of calls reached carousel run considers itself finished.

Details of DecrementingCarousel are specified in decrementing_carousel
